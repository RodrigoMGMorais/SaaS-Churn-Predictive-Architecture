# 📊 Análise e Modelo Preditivo de Churn

Nesta pasta está o notebook com a implementação prática do projeto: **Previsão de Risco de Cancelamento em Clientes SaaS**.

---

## 📌 Conteúdo do arquivo
- ✅ **Carregamento de dados**: Leitura da base de exemplo diretamente do repositório
- 📊 **Análise exploratória**: Entendimento dos dados, padrões e indicadores
- 📈 **Visualizações**: Gráficos que mostram a relação entre uso, suporte e risco de saída
- 🤖 **Modelo preditivo**: Algoritmo de Regressão Logística para calcular a probabilidade de um cliente cancelar
- 📋 **Exemplo prático**: Simulação de risco para um cliente novo

---

## 🛠️ Tecnologias utilizadas
- **Python 3**
- `Pandas` → Manipulação de dados
- `Matplotlib` e `Seaborn` → Criação de gráficos
- `Scikit-learn` → Treinamento e avaliação do modelo de Machine Learning
